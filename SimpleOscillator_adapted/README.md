# SimpleOscillator 
**Author:** Andrew Ikenberry

**Date:** May 2020

**Hardware:** Daisy Pod

## Description: 
Example showing a basic SimpleOscillator with controls.

## Controls:

**Pot 1:** Coarse frequency 10Hz - 12kHz

**SW 1:** Octave down

**SW 2:** Octave up

**Encoder:** Waveform choice (sine, triangle, band-limited sawtooth, band-limited square)
